Recipe Database

The recipes table contains information about each recipe, including the recipe ID, recipe name, cuisine type, cuisine, difficulty, preparation time, and cook time. The ingredients table contains information about each ingredient, including the ingredient ID, ingredient name, and category. The recipe_ingredients table is a linking table that connects recipes with their ingredients, with information about the amount of each ingredient needed for the recipe. The steps table contains information about the steps needed to prepare each recipe. The ratings table contains information about the ratings for each recipe, including the rating ID, recipe ID, rating value, and reviewer name.
